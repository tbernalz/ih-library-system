# IH Library System

A library management system built with .NET 10.0 and Clean Architecture.

## Description

Basic library management system, ready for cloud platforms.

## Tech Stack

- .NET 10.0
- ASP.NET Core
- PostgreSQL
- Entity Framework Core
- Docker
- Open Router API
- Render
- Supabase
